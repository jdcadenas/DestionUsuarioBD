# Gestión de Usuarios con SQLite y Windows Forms (SharpDevelop 4.4)

Proyecto base para aprender a conectar una base de datos SQLite con una aplicación Windows Forms.  
Incluye operaciones CRUD (Crear, Leer, Actualizar, Eliminar) y búsqueda por nombre.

## Requisitos
- SharpDevelop 4.4  (con .NET Framework 4.5)
- Sistema operativo Windows
